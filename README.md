# MeetIQ

MeetIQ is an intelligent meeting assistant and analytics platform designed to capture, process, and extract insights from meeting records.

## Project Structure

The project is structured as follows:

```text
meetiq/
├── backend/            # Backend service (API, processing pipelines, database models)
├── frontend/           # Frontend web application interface
├── sample_meetings/    # Sample audio, transcriptions, and meeting metadata files for testing
├── docs/               # Technical documentation and system designs
├── README.md           # This file
└── .gitignore          # Git exclusion rules
```
